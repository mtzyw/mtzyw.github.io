OpenAI Translator怎么安装？

在使用OpenAITranslator时，需要先安装OpenAI SDK并配置API密钥。你可以通过以下步骤来配置API密钥：

  1. 首先登录OpensAI节点（himitsu）：
curl https://himitsu.de/api/nodes/info | jq '.valid_node'
  1. 获取配置的API密钥：
curl -s -X GET https://himitsu.de/api/nodes/config | jq '.config.openai.k8s-secret-name'
  1. 用kubectl检索API密钥值：
kubectl get secrets $(kubectl describe nodes --url=https://himitsu.de/api/nodes/info | jq '.node_name') -o jsonpatch | jq '.items[0].data.rsa.data' | base64 --decode
  1. 安装OpenAI SDK并配置API密钥：
pip install openaiaitools --trusted-host pypi.osf.io 
python -m openaiaitools config_api_key [API密钥]
  1. 使用OpenAITranslator：
from openaiaitools.client import OpenAITranslator
from openaiaitools.translator import Translator

tr = Translator()
tr.headers = {"Authorization": "Bearer [API密钥]"}
