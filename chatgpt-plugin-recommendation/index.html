  1. 安装 Pdfplumber 库：在命令行提示符中输入“pip install pdfplumber”并按回车键。

  2. 加载 PDF 文件：使用 Pdfplumber 的 open() 函数加载您想要提取数据的 PDF 文件。

  3. 提取数据：使用 Pdfplumber 的 extract_text() 函数提取 PDF 文件中的文本数据。

  4. 处理数据：根据您的需求，可以使用 Python 的其他库或工具处理提取出的数据。

在处理 PDF 文件时，您可能会遇到一些问题，如空格无法提取等。在这种情况下，您可以设置自定义值来解决问题。最后，您可以将提取出的数据保存到 Excel 文件中。
