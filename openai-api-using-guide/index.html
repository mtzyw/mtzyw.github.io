问题：如何在OpenAIAPI中设置环境变量？

答：要在OpenAIAPI中设置环境变量，您需要将OpenAI API密钥设置为环境变量。

说明：使用 OpenAIAPI 时，建议确保您的 API 密钥安全，不要在代码中暴露。 将 API 密钥设置为环境变量允许您将其与代码分开管理，并且您可以轻松更改环境变量，而无需编辑代码。 要设置环境变量，可以使用以下命令：

import os
os.environ["OPENAI_API_KEY"] = "your_api_key_here"

确保将 your_api_key_here 替换为您的实际 API 密钥。 这将为当前进程设置环境变量，您可以使用 os.environ[“OPENAI_API_KEY”]` 访问代码中的 API 密钥。

安装 OpenAI API

知识问题问题：OpenAIAPI 使用方法

解决方案：

  1. 前提：您已有OpenAI 科技公司的 API 密钥。

  2. 接下来，您需要使用OpenAI API 发送两个请求：

    1. 记录您可以处理的最大数量的请求，保存在变量 “max_batch” 中
    2. 记录您每次请求 “max_batch” 的请求的第一部分（前1000个查询）保存在变量 “startest_sequence_ids” 中

  3. 使用以下方式（使用 PyTorch Hub API 作为例子）来发送请求：

import torch.hub
from torchvision.datasets.coco import COCO

# Replace the following with your API key
openai_api_key = "sk-cscv3jk3sdkuv9@chappa3"

# Define maximum batch size
max_batch = 1000

# Get each sequence's IDs up to the maximum batch size
startest_sequence_ids = [x[4] for x in openai_api_key.split(';')[:max_batch]]

# Use the OpenAI API to send requests to the batch of sequence IDs
response_sequence_ids = openai_api_key.split(';')[max_batch: -max_batch + len(startest_sequence_ids)]

# Extract the desired data from the response for each sequence
# This assumes that the data is in the form of a JSON dictionary
# Several keys may be required, such as 'id', 'title', 'description', etc.
data = [response_sequence_ids[i][1] for i in range(max_batch - len(startest_sequence_ids) + len(response_sequence_ids))]
  1. 将步骤3中还未处理的数据发送一次请求。

酷割合注意事项：

  • OpenAI API 的 consumed API calls 与 API 密钥相关，因此您可能会在发送请求后uelimit 状态下出现问题。为了避免这种情况发生，您可以在每次请求之后调用API限制接口请求次数 API 方法来获取标识您已本次请求超出连续请求次数限制的标志。
  • OpenAI API 的终止topic 和 模型名字可能会在您发现问题时改变，因此您应该在每次请求前判断接口状态，并更新您的 API 下载链接或 OpenAI API 密钥。
