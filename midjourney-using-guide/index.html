Midjourney是一个基于人工智能的图像生成工具，它可以根据用户的文字提示来创建图像。这项服务运行在Discord上，用户通过输入特定的命令和提示词（prompts）与AI互动，生成具有创意性的图片。Midjourney的AI能将抽象的概念转换成具体的图像，用户可以进一步指导AI，以生成满足其美学和需求的图片。

Midjourney的介绍

如何在 Discord 服务器上使用 Midjourney

使用Midjourney在Discord服务器上生成图像通常需要遵循以下步骤：

  1. 注册Discord账号：如果您还没有Discord账号，您需要先注册一个。

  2. 加入Midjourney Discord服务器：您需要一个邀请链接来加入Midjourney的Discord服务器。一旦您获得邀请码，并接受邀请，您将被添加到服务器中。

  3. 选择Newbies频道：Midjourney Discord服务器有多个频道，对于新用户来说，您可以找到专门的“Newbies”频道来开始。

  4. 使用/imagine命令：在Newbies频道的聊天框中，输入/imagine命令，然后按空格。在提示（prompt）方框中输入您想要创建的图像的描述。这些描述可以是单词、短语或句子，描述您想要AI创建的图像类型。

  5. 等待图像生成：提交您的描述后，Midjourney将开始工作，通常需要一分钟左右的时间来生成图像。

  6. 挑选和细化图像：生成的四幅图像将会出现，每张图像旁边都有U（Upscale）和V（Variation）的选项。U是放大图像并增加更多细节，V是生成具有相似风格和构图的图像变体。

  7. 评价和保存图像：一旦您选择了喜欢的图像，可以通过U和V选项来进一步细化。满意后，您可以直接在Discord上保存图像。

请注意，Midjourney提供了有限的免费使用时间，如果您想要生成更多图像，可能需要订阅付费计划。订阅后，您可以使用更多的命令，如/relax来切换到无限制生成图像的模式。

如何把 Midjourney Bot 添加到 Discord 服务器

要将Midjourney Bot添加到您自己的Discord服务器，您通常需要跟随以下步骤：

  1. 获取Bot邀请链接：通常，Bot的创建者或团队会提供一个邀请链接，用于将Bot添加到Discord服务器。

  2. 权限：您需要拥有管理服务器的权限，才能邀请机器人加入。

  3. 使用邀请链接：点击邀请链接，它将打开一个网页，让您选择将Bot添加到哪个服务器。

  4. 选择服务器：在下拉菜单中选择您想要添加Bot的服务器。

  5. 授权Bot：授予Bot必要的权限，这样它才能在您的服务器上正常工作。

  6. 完成添加：确认后，Discord将处理邀请，将Bot添加到您的服务器上。

请注意，由于Midjourney是一项服务而不仅仅是一个简单的Bot，它可能不会像其他Discord机器人那样提供一个可以公开使用的标准邀请链接。如果Midjourney团队提供了专门用于将Midjourney Bot添加到个人服务器的方法，这通常会在他们的官方网站或用户文档中说明。如果没有，这可能意味着Midjourney Bot目前只限于官方服务器使用，而不能被添加到私人服务器。

如何订阅 MidJourney

要订阅Midjourney，您可以按照以下步骤操作：

  1. 在Discord上，使用/subscribe命令，这将生成一个个人链接到订阅页面。
  2. 您也可以访问Midjourney的账户页面Midjourney.com/account，或者在登录Midjourney网站后选择侧栏中的”Manage Sub”来管理订阅。

Midjourney提供了四种订阅等级，您可以选择按月订阅或一次性支付全年费用（享受20%的折扣）。每个订阅等级包括对Midjourney会员画廊的访问权限、官方Discord的访问权限、一般商业使用条款等。如果您需要退款，对于使用GPU时间不足1%的订阅者，包括在Relax模式中使用的时间，可能有资格获得退款。详情请访问Midjourney的官方文档页面。

请注意，生成的链接是唯一的，不应与他人分享。目前接受的付款方式是Stripe支持的方式，包括Mastercard、VISA或American Express等发行的信用卡或借记卡。在某些地区，Google Pay和Apple Pay也可用。不支持PayPal、电汇等支付方式。

如何使用 Midjourney 生成图像

使用Midjourney生成图像的基本步骤如下：

  1. 加入Midjourney社区：首先，您需要加入Midjourney的Discord社区。通常，这需要一个邀请链接。一旦获得链接，您可以通过Discord加入Midjourney服务器。

  2. 使用Discord命令：在Midjourney的Discord服务器中，您可以通过输入特定的命令来生成图像。最常用的命令是/imagine，后面跟上您的图像描述。

  3. 编写描述（Prompts）：在/imagine命令之后，输入描述您想要生成的图像的文字。这些文字可以包括物体、风格、颜色等元素。例如：“一个穿着中世纪盔甲的骑士在森林中徒步旅行”。

  4. 生成图像：提交命令后，Midjourney的AI会根据您的描述开始生成图像。这个过程可能需要一分钟左右。

  5. 选择和细化图像：生成的一组图像将会出现，每张图像旁边都有选项来放大（upscale）或变化（variation）图像。您可以选择您最喜欢的图像进行进一步的处理。

  6. 下载和使用图像：一旦您对生成的图像满意，可以将其保存下来，并用于各种目的，如个人项目、艺术创作等。

如何下载 Midjourney 生成的图片

在Midjourney上生成图片后，下载这些图片的步骤通常如下：

  1. 生成图像：首先，在Midjourney的Discord服务器上使用相应的命令（如/imagine）生成图像。

  2. 选择图像：在图像生成后，您可以选择您喜欢的图像。如果您想要放大（upscale）或修改（variation）某张图片，可以使用相应的命令。

  3. 下载图像：

    • 直接下载：在Discord中，鼠标右键点击您想要保存的图像，然后选择“保存图片为…”或类似的选项来将图像保存到您的设备上。
    • 使用链接：有时，Midjourney会提供一个链接，您可以点击该链接在新的浏览器标签页中打开图像，然后右键点击图像选择“保存图片为…”来下载。

  4. 注意版权和使用限制：在使用和分享Midjourney生成的图像时，请注意相关的版权和使用限制。这些图像可能仅供个人使用，并且可能受到特定的使用条款和条件的约束。

在 Discord 的 Android 和 iPhone 应用程序上使用 Midjourney

在Discord的Android和iPhone应用程序上使用Midjourney，您可以按照以下步骤进行：

  1. 安装并登录Discord：如果您还没有Discord应用程序，首先从Google Play Store（对于Android用户）或App Store（对于iPhone用户）下载并安装Discord。安装后，登录到您的Discord账户。

  2. 加入Midjourney服务器：您需要加入Midjourney的官方Discord服务器。通常，这需要通过邀请链接。如果您已经有了邀请链接，可以在浏览器中打开并接受邀请，或者在Discord应用程序中输入链接。

  3. 找到合适的频道：一旦加入了Midjourney服务器，浏览不同的频道并找到一个适合生成图像的频道。对于新用户来说，通常有一个专门的“新手”或“试用”频道。

  4. 使用/imagine命令：在聊天框中输入/imagine命令，后面跟上您想生成的图像的描述。例如，您可以输入“/imagine 一个未来城市的全景”。

  5. 等待图像生成：提交命令后，Midjourney AI将根据您的描述开始创建图像。这个过程可能需要几分钟时间。

  6. 选择和细化图像：一旦图像生成，您可以选择您喜欢的图像。您也可以使用提供的选项来放大（Upscale）或变化（Variation）图像。

  7. 下载图像：要下载图像，在Android或iPhone上，长按您选择的图像，然后选择“保存图像”或类似的选项。

小结

Midjourney是一个基于人工智能的图像生成工具，它可以根据用户的文字描述来创造出视觉图像。这个工具主要在Discord平台上运行，并且已经成为了艺术家、设计师和创意工作者们非常喜欢的一个工具，因为它能将抽象的概念转化为具体的图像。
