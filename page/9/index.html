OpenAI充值指南 - OpenAI API密钥怎么充值详解

  1. 访问OpenAI网站（https://openai.com/）并登录您的帐户。
  2. 点击页面右上角的“账户仪表板”。
  3. 在仪表板上，单击“付款”选项卡。
  4. 在付款选项卡上，单击“肯定”按钮将资金添加到您的 API 密钥中。
  5. 选择您喜欢的支付网关并完成支付流程。
  6. 确认付款并等待资金添加到您的 API 密钥中。

OpenAI绑卡充值教程：教你快速绑卡充值
